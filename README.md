## Performance Solutions

If your computer does not meet the minimum system requirements or is running slowly, you can apply the following tips to improve the program's performance:

1. **Close Unnecessary Applications:** To free up system resources, close any unnecessary programs that may be consuming memory and CPU power.

2. **Disable Windows Visual Effects:**
   - Go to **Control Panel > System > Advanced system settings**.
   - Under **Performance Settings**, choose **Adjust for best performance** or disable certain visual effects like window animations.

3. **Use Lighter Skins:** For improved performance, choose simpler or lighter skins that don’t require a large amount of memory to load.

4. **Optimize Game Settings:**
   - Lower graphical settings in **League of Legends**, such as resolution, texture quality, and other parameters.
   - Turn off extra graphical effects, like shadows and reflections.

5. **System Cooling:** Clean your PC from dust, especially inside the case, to prevent overheating and performance drops.

6. **Update Drivers:** Ensure that your graphics card and processor drivers are up-to-date.

7. **Compatibility Mode:** If the program runs with delays or performance issues, try running it in compatibility mode for an older version of Windows (e.g., Windows 7 or 8).

These tips will help improve performance if your hardware does not meet the recommended system requirements. If performance issues persist, you may need to consider upgrading your computer.
